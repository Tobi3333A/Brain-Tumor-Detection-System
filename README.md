# Brain Tumor Detection System
This is a machine learning model that can tell, wth up to 97% accuracy, whether a brain has tumor from an MRI scan of the brain.

## Overview
With this model, people can tell whether they have a brain tumor and trust that they result they get is most likely accurate because out of 1222 MRI scans tested th this model, only 8 predictions were wrong.

## Tech Stack
- OpenCV
- Scikit-learn
- Support Vector Classifier
